发布日志 Release Note for COS Desktop 0.3, 32-bit PC
发布网址：http://192.168.162.142

1. 介绍
本次是COS Desktop对外发布的第一版，采用基于Linux Mint 15进行二次定制开发，是基于统一脚本自动生成的LiveCD版本。
发布形式采用提供ISO镜像文件下载和已刻录的优盘分发。


2. 主要功能
除了具有基本的Mint系统功能的特点外，主要有以下几个方面。

2.1 显示COS专属标识

2.2 定制软件包
包括：
	新增搜狗中文输入法
	新增wps-office 8.1.0
	新增google-chrome 29.0.1547.65
	新增vim 7.3.547
 	卸载firefox
	卸载LibreOffice

2.3 增加默认字体
默认加入以下字体以便于办公使用
	mtextra.ttf
	symbol.ttf
	WEBDINGS.TTF
	wingding.ttf
	WINGDNG2.ttf
	WINGDNG3.ttf

2.4 系统中文化
默认添加系统中文包，并对开发菜单中的程序进行部分的汉化工作，还有待完成

2.5 定制系统主题、背景等
分别对系统主题、登录界面、开关机界面、安装界面进行了定制。


3. 安装系统
默认仅支持LiveCD格式安装，可将提供的ISO文件刻录到光盘或优盘，再通过已刻录的光盘或优盘进行系统引导安装


4. 更新
本次为第一版本，暂不支持系统更新。之后版本会加入。


5. 更多信息

网址：http://192.168.162.142
