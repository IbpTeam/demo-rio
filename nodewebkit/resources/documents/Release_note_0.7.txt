发布日志 Release Note for COS Desktop 0.7, 32-bit PC

1. 介绍
本次是COS Desktop对外发布的第三版，采用基于Linux Mint 15进行二次定制开发，是基于定制框架自动生成的LiveCD版本。
发布形式采用在线更新、提供公网网站ISO镜像文件下载地址和已刻录的优盘分发三种方式。

2. 本次主要更新

2.1 支持基于COS Desktop 0.5版本的增量更新，更新方式：
确保系统联网，并在终端中执行：
	sudo apt-get update
	sudo apt-get install cos-upgrade
	sudo cos-upgrade -v 0.5 0.7
	待更新结束后，重启即可

2.2 优化LiveCD安装稳定性
替换了更稳定版本的LiveCD内核，修复问题如下：
	a,安装过程随机性无响应，死机情况
	b,安装界面显示比例不协调，“下一步”“上一步”按钮被任务栏覆盖情况
	c,系统安装完成后重启黑屏，死机情况
	d,启动失败，显示initramfs

2.3 更新部分集成软件版本
将QtADB版本从1.0升级到2.0,
	a,优化了整体界面，加入欢迎界面，取消adb调试及命令行模块。
	b,完善了短信功能，重写了手机端apk，增加接受短信功能的稳定性。
	c,增强了对手机环境的适应性，去除Qtadb对busybox的依赖。

2.4 启动界面优化
更换动态启动界面，并加入系统启动进度条显示，实现更好的用户体验

2.5 修复前两版本中帮助信息链接至LinuxMint官网问题
具体如下：
	a,系统设置->语言->帮助链接
	b,系统管理工具->brasero光盘刻录工具->帮助链接
	c,系统管理工具->Gparted磁盘分区工具->帮助链接
	d,系统管理工具->磁盘分析工具->帮助链接
	e,系统管理工具->远程桌面->帮助链接
	f,F1帮助界面链接修复
	g,菜单->帮助链接
	h,附件->帮助链接
	i,附件->阿贴便笺->帮助链接
	j,System Logo帮助链接
	k,欢迎界面上的图标链接
	
2.6 完善系统汉化
具体如下：
	a,登陆窗口选择American English后，menu->applications显示的部分程序名称为中文
	b,菜单->位置->音乐，详细信息为乱码
	c,Gparted解锁界面、应用程序界面为英文
	d,区域与语言设置中英文混杂问题
	e,声音设置中英文混杂问题
	f,welcome界面为英文
	
2.7 修复系统其它功能错误
具体如下：
	a,设置->软件源->输入密码后无法打开问题
	b,设置->更新管理器->安装一个应用程序后“系统设置”变成“系统管理”，并在其目录下只有一个“新立德软件管理器”问题
	c,设置->用户账户，切换普通设置和高级设置多次后账户设置无法打开
	d,“日期和时间设备”，使用网络时间默认勾选，在不输入root密码的情况下可以取消勾选问题
	e,用户登录界面，在非管理员用户下也会显示“您好 Admin”问题
